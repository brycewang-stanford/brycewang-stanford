## 🏗️ What I'm Building

### [StatsPAI](https://github.com/brycewang-stanford/StatsPAI) · `v1.8.0` · 121⭐

**The agent-native Python package for causal inference & econometrics.**
800+ functions across DiD, IV, RD, synthetic control, DML, and causal discovery — unified `.summary()`, `.plot()`, `.to_latex()` API. Brings R's *Causal Inference Task View* and Stata's core econometrics into a single, consistent Python toolkit.

```bash
pip install statspai
```

### [Awesome-Agent-Skills](https://github.com/brycewang-stanford/Awesome-Agent-Skills-for-Empirical-Research) · 521⭐

**23,000+ AI agent skills for empirical research across 8 social science disciplines.**
End-to-end automation: topic selection → literature review → data acquisition → cleaning → analysis → writing → revision. *"20 分钟完成一篇可复现的规范实证论文。"*

### [CoPaper.AI](https://www.copaper.ai) · *Co-founder* · Live

**Your AI Research Co-Author · 从数据到论文，你的共作助手**
A research platform pairing you with an AI co-author across the full paper lifecycle — from raw data to publishable manuscript.
